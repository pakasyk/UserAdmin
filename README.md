# Instructions
### Installation:
1. Clone repository <br />
2. Using terminal: <br />
npm install<br/>
cd views <br />
npm install<br />
3. To start App: <br />
cd .. <br />
npm run dev

### Information:
Front-End runs on create-react-app package port 3000<br />
Back-End runs on Express port 5000

## App
This is User Administrative Site. You can create/edit/delete user.
